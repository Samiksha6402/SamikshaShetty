Welcome to the API Performance & Failure Analysis Dashboard

A real-time monitoring system built to track, analyze, and visualize API activities. This project provides a clear view of API calls, response times, error trends, and usage across regions, methods, and client platforms. With interactive charts and maps, you can quickly spot failures, monitor performance, and gain insights into API usage patterns. Designed with Flask, SQLite, and Chart.js, the dashboard ensures both live updates and powerful filtering options, making it easier to troubleshoot issues and optimize API performance.
